# 目录操作

## 读取目录下文件

```Go
func dirOperation() {
	dirEntry, err := os.ReadDir(".")
	if err != nil {
		fmt.Println("dirEntry err", err)
		return
	}
	for _, entry := range dirEntry {
		fmt.Println(entry.IsDir(), entry.Name())
		fileInfo, _ := entry.Info()
		fmt.Println(fileInfo.Size())
	}
}

func main() {
	dirOperation()
}
```
